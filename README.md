# Python-Programs
This is where I like to keep my python programs. Enjoy
## How to use programs:
### Autoclicker:
S to start <br>
C to stop <br>
While stopped, X to kill program
### Autopaster:
S to start <br>
Put mouse in any corner of your <b><u>main</u></b> screen to kill
### Autotyper
S to start <br>
C to kill
